Getting Started with aPlan4Sheets

From install to your first data pull in under 10 minutes. Follow this step-by-step guide to connect Anaplan to Google Sheets.

In this guide

1 Install from Google Workspace Marketplace 2 Open the Sidebar 3 Connect to Anaplan 4 Configure Your Model 5 Create Your First Pivot 6 Export Anaplan Views 7 Refresh Your Data 8 Tips & Troubleshooting
1

Install from Google Workspace Marketplace

Add aPlan4Sheets to your Google Sheets in one click.

aPlan4Sheets is a Google Sheets add-on available on the Google Workspace Marketplace. It installs directly into your Google Sheets — no desktop software, no downloads, no IT tickets.

  1. Visit the aPlan4Sheets Marketplace listing
  2. Click "Install" and grant the required permissions
  3. Open any Google Sheet — the add-on is now available
Focused walkthrough of the Google Workspace Marketplace listing, install click, and permission handoff before opening Sheets.

Google Workspace Admin? You can install aPlan4Sheets domain-wide from the Admin Console, so every user in your organization has access automatically.

2

Open the Sidebar

Launch the aPlan4Sheets interface inside Google Sheets.

Once installed, the connector lives in your Extensions menu. Open it to reveal the sidebar where all the magic happens.

  1. Open any Google Sheet
  2. Go to Extensions → Anaplan Connector for Sheets™ → 📊 Open Sidebar
  3. If needed, you can also access 🔑 Login to Anaplan and 🔧 Setup OAuth Credentials from the same menu
  4. The aPlan4Sheets sidebar will appear on the right side of your spreadsheet
Shows the Extensions menu path and the sidebar opening inside Google Sheets.

Tip: The sidebar will remember your last session. If you've previously logged in, you may already be connected when you reopen it.

3

Connect to Anaplan

Choose a Login Method, then follow the path that matches your account.

The sidebar opens on a login-first screen with a Login Method selector. Start there, then use the branch below that matches your account setup.

Default

Option A: OAuth/SSO

This is the default Login Method. Most work accounts should use this path.

  1. Leave Login Method set to OAuth/SSO
  2. Click "Login with Anaplan SSO"
  3. A popup window will open so you can complete your organization's SSO login
  4. After successful authentication, return to the sidebar and continue setup

Option B: Gmail or personal-domain account

If you are using Gmail or another personal domain, SSO may need one extra setup step before login will work.

  1. Open Extensions → Anaplan Connector for Sheets™ → 🔧 Setup OAuth Credentials
  2. Follow the guided setup shown in the sidebar for personal-domain access
  3. Return to the login screen and click "Login with Anaplan SSO"
  4. If your Anaplan tenant uses a different authentication pattern, follow your organization's instructions
Covers the login-first sidebar flow and OAuth setup sequence.

Security note: For the basic-auth path, credentials are encrypted and stored securely in Google infrastructure. OAuth/SSO continues to use Anaplan's official authentication flow.

4

Configure Your Model

Select which Anaplan workspace and model to connect.

After logging in, the empty state reads "No models configured yet. Click \"Add Model\" to get started." Add at least one model before using exports, saved views, or pivot views.

  1. Click the "Add Model" button in the sidebar
  2. Select your Workspace from the dropdown — all workspaces you have access to will appear
  3. Select the Model from the second dropdown
  4. Click "Save" to add the model to your configuration
Demonstrates adding a workspace and model after authentication.

Note: You can add multiple models, but there is a known issue where the export list may not load correctly when several models are configured. If exports do not populate, try validating with a single selected model first.

5

Create Your First Pivot

Use Pivot Views to build a custom pivot layout.

The Pivot Views section lets you pull data from an Anaplan view and arrange dimensions exactly how you need them, including rows, columns, and page filters.

  1. Open the "Pivot Views" section in the sidebar
  2. Search or browse the available views in your connected model
  3. Select a view and click "Configure Pivot"
  4. Drag dimensions to arrange them:
    • Rows — dimensions that appear as row headers
    • Columns — dimensions that appear as column headers
    • Pages — dimensions used as filters (select specific items)
  5. Select items for each page dimension to filter your data
  6. Use "Repivot Current Sheet" later if you want to rerun the current pivot with updated settings
Walkthrough of selecting a Pivot View and configuring the pivot layout.

Tip: The pivot layout mirrors what you'd see in Anaplan's own UI. If you've used page selectors in Anaplan, you'll feel right at home.

6

Run Exports and Saved Views

Use the current sidebar labels to pull exports or saved views into Sheets.

The sidebar has separate Exports and Saved Views sections. Use the one that matches the object you want to run.

  1. Open the "Exports" section and use "Search exports..." if you need to filter the list
  2. Select one or more exports, then click "Refresh Selected Exports"
  3. For saved views, switch to the "Saved Views" section
  4. Select the view you want and click "Export View"
Example of browsing model content and running data pulls from the sidebar.

Tip: If exports do not populate after adding more than one model, retry with a single configured model first. Saved Views and Pivot Views may be a better fallback while the multi-model export issue remains unresolved.

7

Refresh Your Data

Update all your configured sheets with fresh Anaplan data.

Once you've set up your exports, saved views, or pivot views, you can refresh everything from the current spreadsheet without rebuilding each connection.

  1. Click "Refresh Sheets" in the sidebar
  2. To automate updates, open "Scheduled Refreshes" and click "+ Add Schedule"
  3. Choose the scope, such as "All Configured Sheets", then click "Save Schedule"
  4. Use "Email Notification Settings" if you want schedule-related email updates
Shows the refresh flow and scheduled refresh configuration pattern.

Auto-refresh: Scheduled refreshes are configured directly in the sidebar under "Scheduled Refreshes", so you can keep reports updated without reopening each setup flow.

Tips & Troubleshooting

Common questions and quick fixes.

Session expired / token error

Anaplan authentication tokens expire after a period of inactivity. If you see an authentication error, simply log in again using the same method (SSO or email/password). Your configurations are preserved — you won't lose any setup.

"Unverified app" warning screen

When first authorizing the add-on, Google may show an "unverified app" warning. This is normal during the verification process. Click "Advanced" at the bottom left, then click "Go to Anaplan Connector for Sheets™ (unsafe)" to proceed. The app is safe — this screen appears for all new Workspace add-ons.

Slow first load

The first time you open the sidebar or load a model's modules, it may take a few extra seconds. This is normal — aPlan4Sheets is fetching metadata from Anaplan's API. Subsequent loads will be faster as the data is cached locally.
